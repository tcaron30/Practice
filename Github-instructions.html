GitHub: repos & interactives

A step-by-step guide for course authors

Step 1 of 6

Create a GitHub account

If you already have one, skip ahead to step 2.

  1. Go to github.com and click Sign up
  2. Choose a username, email, and password
  3. Verify your email address
  4. Select the Free plan
Your username becomes part of your URL: github.com/your-username

Create a new repository

A repository (repo) is a hosted folder for your project.

  1. Click the + icon → New repository
  2. Name your repo (e.g. my-interactives)
  3. Set visibility to Public — required for free Pages hosting
  4. Check Add a README file
  5. Click Create repository
A public repo lets GitHub host your files free at username.github.io/repo-name

Clone the repo to your computer

Download the repo so you can add files locally. You'll need Git installed.

git clone https://github.com/YOUR-USERNAME/my-interactives.git cd my-interactives
No terminal? Use GitHub Desktop (desktop.github.com) for a point-and-click alternative.

After cloning, your folder looks like:

my-interactives/ README.md ← auto-created

Add your interactive files

Each interactive is a self-contained .html file — inline CSS and JS, no external dependencies.

my-interactives/ README.md index.html ← landing page demos/ bubble-sort.html tcp-explainer.html compound-interest.html
Self-contained files are safest — nothing breaks when files move or the folder structure changes.

Commit and push to GitHub

Save your changes and upload them with three commands. Repeat every time you update a file.

git add . # stage all new/changed files
git commit -m "Add interactive demos" # snapshot
git push origin main # upload to GitHub
After pushing, your files appear on github.com within seconds.

Enable GitHub Pages

Turn your repo into a live website — free hosting, instant deploys on every push.

  1. On your repo page, click Settings
  2. Scroll to Pages in the left sidebar
  3. Under Source, select Deploy from a branch
  4. Choose branch: main  /  folder: / (root)
  5. Click Save
After ~60 seconds your site is live at:
https://YOUR-USERNAME.github.io/my-interactives/
Use this URL in Coursera's iframe plugin.
